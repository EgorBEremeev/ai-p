# AI-P: Модель Искусственного субъекта

Проект **AI-P** посвящен разработке и формализации модели искусственного субъекта.

Искусственные субъекты - системы с субъективным опытом, целостным поведением, способностью к дарвиновской эволюции, в которых языковая модель выступает вычислительной семантической средой.
