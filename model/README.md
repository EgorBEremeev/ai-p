# AI-P: Концептуальная и параметрическая модель

В данном каталоге содержится концептуальный базис проекта AI-P.

## Содержимое

| Файл | Описание |
|------|----------|
| **[mdl_AI-P.md](./mdl_AI-P.md)** | **Интегральная концептуальная модель.** Основной документ, описывающий онтологию, пятислойную архитектуру (L1-LM) и цикл когнитивного акта. |
| **[params_AI-P.yaml](./params_AI-P.yaml)** | **Параметрическая модель.** Формальное YAML-описание всех доступных параметров субъекта, используемое для конфигурации конкретных реализаций. |
| **[AI-P_basic_models_family.md](./AI-P_basic_models_family.md)** | **Семейства моделей (A-F).** Классификация искусственных субъектов по уровням автономности и глубине субъектности (Реактивные автоматы, Ретроспективные, Рефлексивные и т.д.). |
| **[decision_table_TEMPLATE_AI-P.md](./decision_table_TEMPLATE_AI-P.md)** | **Шаблон-конфигуратор (Decision Table).** Инструмент для фиксации конкретного набора параметров при проектировании частной архитектуры. |
| **[mdl_AI-P.drawio.svg](./mdl_AI-P.drawio.svg)** | Графическая схема концептуальной модели. |
